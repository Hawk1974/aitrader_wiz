# Desktop Included Files

This Desktop package intentionally includes only canonical runtime files:

- root Alvin runtime markdown files
- the live `claw3d-runtime.json` export
- canonical agent profile files for the AlTrader agents
- Desktop support scripts still relevant to Office and runtime alignment
- Desktop agent reference docs from the repo

This Desktop package intentionally excludes:

- caches
- logs
- databases
- backups
- PID files
- local secrets
- temporary output
- stale session files
- generated cron output
